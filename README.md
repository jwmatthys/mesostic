mesostic
========

Generate automatic mesostics from text files

#TO USE

python mesostic.py

#INSTALL

Requires Tkinter and nltk

on Ubuntu:
sudo apt-get install python-tk python-nltk

on OSX:
install nltk using the instructions here: http://nltk.org/install.html
